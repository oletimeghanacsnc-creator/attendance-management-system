📋 Excel File Format Requirements

Your Excel file should have the following columns in the first sheet:

  • ClassName or Class - The class name (e.g., "CSE")
  • Section - The section (e.g., "A", "B")
  • Subject - The subject name
  • USN - Unique Student Number
  • StudentName or Name - Student's full name

⚠️ Important: Each class can have a maximum of 60 students. If a class reaches this limit, additional students will be rejected.

💡 Tip: The system will automatically create classes if they don't exist, and update student information if the USN already exists.
